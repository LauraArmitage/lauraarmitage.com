Final project - Test-Only Development: I explored the idea of automating the coding step of the red-green-refactor cycle. I created a tool which when given a failing test, amended simple code such that the test would then pass. Although my tool only dealt with basic cases, my conclusion was that given more time and research it would be possible to develop a more extensive version of the tool which could then lead towards automating other stages of the red-green-refactor cycle. The tool would also be valuable in demonstrating how simple the coding step should be when teaching test-driven development, which I tested out on a small sample of my peers.
